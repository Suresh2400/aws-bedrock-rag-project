# AWS Bedrock RAG Application

This project implements a Retrieval-Augmented Generation (RAG) pipeline using AWS Bedrock, LangChain, FAISS, and Streamlit.

## Features
- PDF document ingestion
- Vector embeddings
- Semantic search
- Claude/Titan model integration
- Streamlit chatbot UI

## Technologies
- Python
- LangChain
- AWS Bedrock
- FAISS
- Streamlit
- Boto3
